IP protocol numbers

These Internet Protocol numbers can be found in the Protocol field of the IPv4 header and the Next Header field of the IPv6 header (both are 8-bit long). The same values are used in both versions and define layout of the header that immediately follows IPv4/IPv6 header. These numbers are managed and assigned by the Internet Assigned Numbers Authority (IANA). See full list in Wikipedia.
