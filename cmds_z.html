zip

is a compression and file packaging utility compatible with classical DOS pkzip. There is also a companion program unzip.

zip [-options] [-b path] [-t mmddyyyy]
[-n suffixes] [zipfile list] [-xi list]

The default action is to add or replace zipfile entries from list, which can include the special name '-' to compress stdin. If zipfile and list are omitted, ~ compresses stdin to stdout.

zip last *

put all files in the current dir into archive last.zip;

zip home_dir .* *

put all files (including hidden) in the current dir into compressed archive home_dir.zip;

zip -r proj1-dir proj1

zip up an entire dir proj1 recursively; zipfile name would be proj1-dir.zip;

zip -j msgs /opt/news/*

put all files in the specified dir (/opt/news) into archive msgs.zip removing paths from the file names;

zip -r proj1 proj1 –x tmp

zip up an entire dir proj1 recursively excluding subdir tmp;

zip -r -n mp3 misc tmp2

zip up an entire dir tmp2 recursively, however do not compress mp3 files, just add them to the archvive;

find /home/alex/ -newer tstmp.dat -print |
zip -j backup -@

find and compress all files in /home/alex that are newer than tstmp.dat; the archive name would be backup.zip;

Options

-?    -h    --help    -h2    --more-help    --verbose

-0 (zero) do not compress, just store all files;
-n set compression ratio; n must be in the range 1..9 (from max speed to max compression), default is 6;
-@ read names from stdin;
-b use the specified path for the temporary archive;
-c add one-line comment; you'll be prompted to enter comment for each file (press ENTER for "no comment");
-d delete entries in zipfile;
-D do not add dir entries (equiv to -x "*/");
-e encrypt (you’ll be prompted for password);
-F fix zipfile; you can also use -FF which means "try harder";
-f freshen archive (add only changed files, ignore new);
-i include only the specified names;
-J junk zipfile prefix;
-j junk mode (strip dir names);
-l convert LF to CR+LF (-ll does the opposite);
-m move files into the zip archive, delete sources;
-n do not compress the specified suffixes;
-o make zipfile as old as the latest entry;
-q quiet mode;
-r recursive mode (include subdirs);
-T test zipfile integrity;
-u update archive (add new and changed files);
-v (without other args) version + misc internal info; otherwise it works like --verbose;
-X eXclude eXtra file attributes;
-x exclude the specified names;
-y store symlinks as symlinks instead of the referenced files;
-z add one multi-line comment for the whole archive (you'll be prompted); to finish comment put a period (.) at the new line and press ENTER (CTRL+D also works);
